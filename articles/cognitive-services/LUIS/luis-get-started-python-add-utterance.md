### <a name="add-utterances-from-the-command-line"></a>명령줄에서 발언 추가

인수 없이 add-utterance를 호출하면 발언을 학습하지 않고 앱에 추가합니다.

```console
> python add-utterances-3-6.py
```

모델에 발언이 추가되면 다음 응답이 반환됩니다.  

