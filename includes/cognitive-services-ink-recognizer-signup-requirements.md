Ink Recognizer API에 대한 액세스 권한이 있는 [Cognitive Services API 계정](../articles/cognitive-services/cognitive-services-apis-create-account.md)이 있어야 합니다. Azure 구독이 없는 경우 [무료 계정](https://azure.microsoft.com/try/cognitive-services/)을 만들 수 있습니다. 구독 키는 계정을 만든 후 또는 [Azure Portal](../articles/cognitive-services/cognitive-services-apis-create-account.md#access-your-resource)에서 또는 평가판을 활성화한 후 [Azure 웹 사이트](https://azure.microsoft.com/try/cognitive-services/my-apis)에서 구할 수 있습니다.
